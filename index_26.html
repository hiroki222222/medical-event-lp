About the Event

医療職のための
最大規模フォーラム

MedConnect Forumは、医療・介護・福祉の現場で働くプロフェッショナルが集い、 キャリアアップ・知識の更新・人脈形成を図るための総合的なイベントです。 第7回となる今回は、AI医療・地域医療・多職種連携をテーマに、 第一線で活躍する専門家によるセッションをお届けします。

  • 医師・看護師・薬剤師・理学療法士など全職種対象
  • 継続的専門能力開発（CPD）対応プログラムを多数収録
  • 100社以上の医療関連企業が出展する展示エリア
  • 懇親会・名刺交換会による充実したネットワーキング
Program

充実の6大プログラム

あらゆるキャリアステージの医療職に対応した多彩なコンテンツをご用意しています。

Program 01
最新医療トピック
基調講演

AI診断・遺伝子医療・デジタルヘルスなど、今最も注目される医療技術の現在と未来を第一線の専門家が解説します。

Program 02
キャリアデザイン
ワークショップ

専門医・認定看護師・管理職転換など、あなたのキャリアパスを具体化するインタラクティブなワークショップです。

Program 03
多職種連携
パネルディスカッション

医師・看護師・薬剤師・リハビリ職が同じテーブルで議論。チーム医療の実践知を現場目線で共有します。

Program 04
地域医療・
へき地医療セッション

地域包括ケアシステムの最前線に立つ医療者が、地方・離島・在宅での実践事例を報告します。

Program 05
医療スタートアップ
ピッチイベント

医療現場の課題を解決する次世代スタートアップが集結。革新的なソリューションをライブで体感してください。

Program 06
メンタルヘルス&
ウェルネスゾーン

医療者のバーンアウト予防・セルフケア実践法をテーマに、専門家によるミニセッションと交流会を開催します。

Why Attend

このフォーラムに
参加すべき理由

単なる勉強会ではありません。医療の現在と未来を同じ熱量で語れる仲間と出会い、 あなたのキャリアと医療現場を変えるきっかけがここにあります。

1,200名以上

が参加した医療職のコミュニティ

実践的な知識

教科書には載っていない、現場で今すぐ活かせる最新エビデンスと事例を提供します。

横断的な人脈

診療科・職種・地域を超えた医療人のネットワークが、あなたの視野と可能性を広げます。

キャリアへの投資

転職・昇進・独立開業など、次のステップを具体的に考える機会をプログラムで設けています。

グローバル視点

海外の医療制度・研究動向を学べるインターナショナルセッションを今年も開催します。

主催団体について
